### Title
Missing Receiver Address Validation Allows Permanent NFT Locking via Escrow Address as Receiver — (`x/nft-transfer/keeper/packet.go`)

### Summary
`processReceivedPacket` does not validate that the `receiver` address in an incoming IBC packet is not an escrow address. An attacker holding an IBC voucher NFT (sink-chain direction) can burn the voucher and embed the origin chain's own escrow address as the receiver, causing the origin chain to "unescrow" the NFT back to the escrow address itself, permanently locking it.

### Finding Description
In `processReceivedPacket`, when `isAwayFromOrigin=false` (return-to-origin direction), the function calls:

```go
k.nftKeeper.TransferOwner(ctx, voucherClassID, tokenID, escrowAddress, receiver)
```

where `receiver` is taken directly from `data.Receiver` with no check that it differs from `escrowAddress` or that it is not a module/escrow account. Neither `MsgTransfer.ValidateBasic()` nor `NonFungibleTokenPacketData.ValidateBasic()` nor the ante handler perform any such check. The escrow address for any port/channel pair is deterministic and publicly queryable.

### Impact Explanation
The attacker's voucher NFT on Chain B is burned (irreversible). On Chain A, `TransferOwner` is called with `from = to = escrowAddress`, leaving the NFT in the escrow address with no user able to claim it. The NFT is permanently destroyed from the user's perspective. A variant attack sets `receiver` to a *different* channel's escrow address on Chain A (e.g., `GetEscrowAddress("nft", "channel-C")`), which moves the NFT out of `escrowAddress_channelA` into `escrowAddress_channelC`, corrupting the escrow invariant for channel-C.

### Likelihood Explanation
Any holder of an IBC voucher NFT can execute this. The escrow address is publicly queryable. No special privileges, governance, or operator access is required. The attack is a single signed `MsgTransfer` transaction followed by normal relaying.

### Recommendation
In `processReceivedPacket`, after parsing `receiver`, add a guard:

```go
if receiver.Equals(escrowAddress) {
    return sdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "receiver cannot be the escrow address")
}
```

More broadly, validate in `MsgTransfer.ValidateBasic()` and/or `processReceivedPacket` that the receiver is not any known escrow address (or at minimum not the escrow address for the receiving channel). The ICS-20 fungible token transfer module in ibc-go addresses this with a `BlockedAddr` check against the bank module's blocked addresses; an analogous mechanism should be added here.

### Proof of Concept
1. Chain A has native NFT class `nftClass`, token `token1`, owned by Alice.
2. Alice transfers to Chain B via `(nft, channel-A)` → `(nft, channel-B)`. Chain A escrows `token1` in `escrowAddr_A = GetEscrowAddress("nft", "channel-A")`. Chain B mints voucher `ibc/<hash>/token1` to Alice.
3. Attacker (Alice) queries `escrowAddr_A` via `GET /ibc/apps/nft_transfer/v1/channels/channel-A/ports/nft/escrow_address`.
4. Attacker submits `MsgTransfer{SourcePort:"nft", SourceChannel:"channel-B", ClassId:"ibc/<hash>", TokenIds:["token1"], Sender: alice, Receiver: escrowAddr_A.String(), ...}` on Chain B.
5. `createOutgoingPacket`: `isAwayFromOrigin=false` → `BurnNFTUnverified` burns the voucher. Packet sent with `receiver=escrowAddr_A`.
6. Relayer delivers packet to Chain A. `processReceivedPacket`: `isAwayFromOrigin=false` → `TransferOwner(ctx, "nftClass", "token1", escrowAddr_A, escrowAddr_A)`.
7. Assert: `nftKeeper.GetNFT("nftClass", "token1").GetOwner() == escrowAddr_A`. No user holds the voucher on Chain B. No user can claim `token1` on Chain A. NFT is permanently locked.

func (nftpd NonFungibleTokenPacketData) ValidateBasic() error {
	if strings.TrimSpace(nftpd.ClassId) == "" {
		return newsdkerrors.Wrap(ErrInvalidClassID, "classId cannot be blank")
	}

	if len(nftpd.TokenIds) == 0 {
		return newsdkerrors.Wrap(ErrInvalidTokenID, "tokenId cannot be blank")
	}

	if len(nftpd.TokenIds) != len(nftpd.TokenUris) {
		return newsdkerrors.Wrap(ErrInvalidPacket, "tokenIds and tokenUris lengths do not match")
	}

	if strings.TrimSpace(nftpd.Sender) == "" {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "sender address cannot be blank")
	}

	// decode the sender address
	if _, err := sdk.AccAddressFromBech32(nftpd.Sender); err != nil {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "invalid sender address")
	}

	if strings.TrimSpace(nftpd.Receiver) == "" {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "receiver address cannot be blank")
	}

	// decode the receiver address
	if _, err := sdk.AccAddressFromBech32(nftpd.Receiver); err != nil {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "invalid receiver address")
	}
	return nil

func (vtd ValidateMsgTransferDecorator) AnteHandle(ctx sdk.Context, tx sdk.Tx, simulate bool, next sdk.AnteHandler) (sdk.Context, error) {
	// avoid breaking consensus
	if !ctx.IsCheckTx() {
		return next(ctx, tx, simulate)
	}

	msgs := tx.GetMsgs()
	for _, msg := range msgs {
		transfer, ok := msg.(*nfttypes.MsgTransfer)
		if !ok {
			continue
		}

		if len(transfer.ClassId) > MaxClassIDLength {
			return ctx, newsdkerrors.Wrapf(sdkerrors.ErrInvalidRequest, "class id length must be less than %d", MaxClassIDLength)
		}

		if len(transfer.TokenIds) > MaxTokenIds {
			return ctx, newsdkerrors.Wrapf(sdkerrors.ErrInvalidRequest, "token id length must be less than %d", MaxTokenIds)
		}

		for _, tokenID := range transfer.TokenIds {
			if len(tokenID) > MaxTokenIDLength {
				return ctx, newsdkerrors.Wrapf(sdkerrors.ErrInvalidRequest, "token id length must be less than %d", MaxTokenIDLength)
			}
		}

		if len(transfer.Receiver) > MaximumReceiverLength {
			return ctx, newsdkerrors.Wrapf(sdkerrors.ErrInvalidRequest, "receiver length must be less than %d", MaximumReceiverLength)
		}
	}

	return next(ctx, tx, simulate)

func GetEscrowAddress(portID, channelID string) sdk.AccAddress {
	// a slash is used to create domain separation between port and channel identifiers to
	// prevent address collisions between escrow addresses created for different channels
	contents := fmt.Sprintf("%s/%s", portID, channelID)

	// ADR 028 AddressHash construction
	preImage := []byte(Version)
	preImage = append(preImage, 0)
	preImage = append(preImage, contents...)
	hash := sha256.Sum256(preImage)
	return hash[:20]
