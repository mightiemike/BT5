### Title
NFT Permanently Non-Burnable After Transfer to Non-Creator — (`x/nft/keeper/keeper.go`)

### Summary

`BurnNFT` enforces two independent authorization checks in sequence: `IsOwner` **AND** `IsDenomCreator`. After a denom creator transfers an NFT to any non-creator address, no address in the system can satisfy both checks simultaneously, making the token permanently non-burnable.

### Finding Description

`BurnNFT` in `keeper.go` requires the caller to be both the current NFT owner and the denom creator: [1](#0-0) 

`IsOwner` checks that `owner == nft.Owner` (the current holder): [2](#0-1) 

`IsDenomCreator` checks that `owner == denom.Creator`, a field that is set once at `IssueDenom` and never updated: [3](#0-2) 

`TransferOwner` only requires `IsOwner` (no creator check), so the denom creator can freely transfer any NFT to a non-creator: [4](#0-3) 

After the transfer:
- **B (new owner)** calls `BurnNFT`: `IsOwner` passes, `IsDenomCreator` fails → `ErrUnauthorized`
- **A (denom creator)** calls `BurnNFT`: `IsOwner` fails (A is no longer owner) → `ErrUnauthorized`

The token ID is permanently locked. `BurnNFTUnverified` (which skips the creator check) exists but is only called internally for IBC escrow/burn flows — it is not reachable via `MsgBurnNFT`: [5](#0-4) 

The `MsgBurnNFT` handler routes exclusively through `k.Keeper.BurnNFT`: [6](#0-5) 

### Impact Explanation

Any NFT transferred from its denom creator to a non-creator address becomes permanently non-burnable. The token ID is locked in state forever: it cannot be destroyed, and since `MintNFT` rejects duplicate token IDs, the ID cannot be reused. This is a direct violation of the invariant that an NFT owner should be able to destroy their own asset. [7](#0-6) 

### Likelihood Explanation

This is triggered by the normal, intended workflow: issue a denom, mint an NFT to a recipient (or mint to self and transfer), then attempt to burn. The `MsgMintNFT` message explicitly accepts a `Recipient` field distinct from `Sender`, so minting directly to a non-creator is a first-class supported operation. No special privileges or governance are required. [8](#0-7) 

### Recommendation

Remove the `IsDenomCreator` check from `BurnNFT`. Burning should require only ownership. If the intent is to restrict burning to the creator, that policy must be documented and `TransferOwner` must be blocked or the creator check must be removed. The simplest correct fix is to replace the body of `BurnNFT` with the logic already present in `BurnNFTUnverified` (owner-only check), or to call `BurnNFTUnverified` directly from `MsgBurnNFT`. [1](#0-0) 

### Proof of Concept

Reproducible as a keeper unit test (no mocks needed):

```
1. IssueDenom(ctx, "denom1", ..., addrA)          // A is creator
2. MintNFT(ctx, "denom1", "token1", ..., addrA, addrA)  // A mints to self
3. TransferOwner(ctx, "denom1", "token1", addrA, addrB)  // A transfers to B
4. BurnNFT(ctx, "denom1", "token1", addrB)
   → IsOwner: PASS (B is owner)
   → IsDenomCreator: FAIL (B is not creator)
   → returns ErrUnauthorized ✓
5. BurnNFT(ctx, "denom1", "token1", addrA)
   → IsOwner: FAIL (A is no longer owner)
   → returns ErrUnauthorized ✓
```

Both callers are permanently blocked. The NFT is irrecoverably locked in state. [9](#0-8)

```go
func (k Keeper) MintNFTUnverified(ctx sdk.Context, denomID, tokenID, tokenNm, tokenURI, tokenData string, owner sdk.AccAddress) error {
	if !k.HasDenomID(ctx, denomID) {
		return sdkerrors.Wrapf(types.ErrInvalidDenom, "denom ID %s not exists", denomID)
	}

	if k.HasNFT(ctx, denomID, tokenID) {
		return sdkerrors.Wrapf(types.ErrNFTAlreadyExists, "NFT %s already exists in collection %s", tokenID, denomID)
	}

	k.setNFT(
		ctx, denomID,
		types.NewBaseNFT(
			tokenID,
			tokenNm,
			owner,
			tokenURI,
			tokenData,
		),
	)
	k.setOwner(ctx, denomID, tokenID, owner)
	k.increaseSupply(ctx, denomID)

	return nil
}

// MintNFT mints an NFT and manages the NFT's existence within Collections and Owners
func (k Keeper) MintNFT(
	ctx sdk.Context, denomID, tokenID, tokenNm,
	tokenURI, tokenData string, sender, owner sdk.AccAddress,
) error {
	_, err := k.IsDenomCreator(ctx, denomID, sender)
	if err != nil {
		return err
	}

	return k.MintNFTUnverified(ctx, denomID, tokenID, tokenNm, tokenURI, tokenData, owner)
}
```

```go
func (k Keeper) IsOwner(ctx sdk.Context, denomID, tokenID string, owner sdk.AccAddress) (types.BaseNFT, error) {
	nft, err := k.GetNFT(ctx, denomID, tokenID)
	if err != nil {
		return types.BaseNFT{}, err
	}

	if !owner.Equals(nft.GetOwner()) {
		return types.BaseNFT{}, sdkerrors.Wrapf(types.ErrUnauthorized, "%s is not the owner of %s/%s", owner.String(), denomID, tokenID)
	}

	return nft.(types.BaseNFT), nil
}
```

```go
func (k Keeper) IsDenomCreator(ctx sdk.Context, denomID string, address sdk.AccAddress) (types.Denom, error) {
	denom, err := k.GetDenom(ctx, denomID)
	if err != nil {
		return types.Denom{}, err
	}

	creator, err := sdk.AccAddressFromBech32(denom.Creator)
	if err != nil {
		panic(err)
	}

	if !creator.Equals(address) {
		return types.Denom{}, sdkerrors.Wrapf(types.ErrUnauthorized, "%s is not the creator of %s", address, denomID)
	}

	return denom, nil
```

```go
func (m msgServer) MintNFT(goCtx context.Context, msg *types.MsgMintNFT) (*types.MsgMintNFTResponse, error) {
	recipient, err := sdk.AccAddressFromBech32(msg.Recipient)
	if err != nil {
		return nil, err
	}

	sender, err := sdk.AccAddressFromBech32(msg.Sender)
	if err != nil {
		return nil, err
	}

	ctx := sdk.UnwrapSDKContext(goCtx)
	if err := m.Keeper.MintNFT(ctx, msg.DenomId, msg.Id,
		msg.Name,
		msg.URI,
		msg.Data,
		sender,
		recipient,
	); err != nil {
		return nil, err
	}

	ctx.EventManager().EmitEvents(sdk.Events{
		sdk.NewEvent(
			types.EventTypeMintNFT,
			sdk.NewAttribute(types.AttributeKeyTokenID, msg.Id),
			sdk.NewAttribute(types.AttributeKeyDenomID, msg.DenomId),
			sdk.NewAttribute(types.AttributeKeyTokenURI, msg.URI),
			sdk.NewAttribute(types.AttributeKeyRecipient, msg.Recipient),
		),
		sdk.NewEvent(
			sdk.EventTypeMessage,
			sdk.NewAttribute(sdk.AttributeKeyModule, types.AttributeValueCategory),
			sdk.NewAttribute(sdk.AttributeKeySender, msg.Sender),
		),
	})

	return &types.MsgMintNFTResponse{}, nil
}
```

```go
func (m msgServer) BurnNFT(goCtx context.Context, msg *types.MsgBurnNFT) (*types.MsgBurnNFTResponse, error) {
	sender, err := sdk.AccAddressFromBech32(msg.Sender)
	if err != nil {
		return nil, err
	}

	if err := m.Keeper.BurnNFT(ctx, msg.DenomId, msg.Id, sender); err != nil {

			sdk.EventTypeMessage,
			sdk.NewAttribute(sdk.AttributeKeyModule, types.AttributeValueCategory),
			sdk.NewAttribute(sdk.AttributeKeySender, msg.Sender),
		),
```
