### Title
`ValidResourceBounds` Protobuf Round-Trip Silently Collapses `AllResources` to `L1Gas`, Producing a Divergent Transaction Hash — (`crates/apollo_protobuf/src/converters/transaction.rs`)

---

### Summary

The `TryFrom<protobuf::ResourceBounds> for ValidResourceBounds` conversion uses a value-based heuristic: if both `l2_gas` and `l1_data_gas` are zero it emits `ValidResourceBounds::L1Gas`, otherwise `ValidResourceBounds::AllResources`. A transaction submitted with `AllResources { l1_gas: X, l2_gas: 0, l1_data_gas: 0 }` is accepted by the gateway, has its hash computed under the `AllResources` branch (three resource felts), is stored in a block as `InvokeTransactionV3 { resource_bounds: ValidResourceBounds::AllResources(…) }`, and then — when that block is serialised to protobuf and deserialised on a peer — is silently re-classified as `L1Gas(X)` (two resource felts). `get_tip_resource_bounds_hash` produces a strictly different Poseidon digest for the two variants, so the hash bound to the transaction on the originating node differs from the hash recomputed on any node that deserialises the block from protobuf.

impl TryFrom<protobuf::ResourceBounds> for ValidResourceBounds {
    type Error = ProtobufConversionError;
    fn try_from(value: protobuf::ResourceBounds) -> Result<Self, Self::Error> {
        let Some(l1_gas) = value.l1_gas else {
            return Err(missing("ResourceBounds::l1_gas"));
        };
        let Some(l2_gas) = value.l2_gas else {
            return Err(missing("ResourceBounds::l2_gas"));
        };
        // TODO(Shahak): Assert data gas is not none once we remove support for 0.13.2.
        let l1_data_gas = value.l1_data_gas.unwrap_or_default();
        let l1_gas: ResourceBounds = l1_gas.try_into()?;
        let l2_gas: ResourceBounds = l2_gas.try_into()?;
        let l1_data_gas: ResourceBounds = l1_data_gas.try_into()?;
        Ok(if l1_data_gas.is_zero() && l2_gas.is_zero() {
            ValidResourceBounds::L1Gas(l1_gas)
        } else {
            ValidResourceBounds::AllResources(AllResourceBounds { l1_gas, l2_gas, l1_data_gas })
        })
    }
