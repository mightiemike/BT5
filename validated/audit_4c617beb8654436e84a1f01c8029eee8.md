### Title
NLP Burn Fee Floor Silently Reduces Returned Quote to Zero, Causing Total Position Loss — (`core/contracts/Clearinghouse.sol`)

---

### Summary

In `Clearinghouse.burnNlp()`, a hardcoded minimum `burnFee` of `ONE` ($1 in X18 fixed-point) can consume the entire `quoteAmount` when a user burns NLP tokens worth ≤ $1. The NLP balance is fully decremented while the user receives zero quote tokens in return. This is a direct analog to M02's `redemptionTax` finding: a fee mechanism silently reduces the returned asset amount to zero with no revert or warning.

---

### Finding Description

In `Clearinghouse.burnNlp()`:

```solidity
int128 quoteAmount = nlpAmount.mul(oraclePriceX18);
int128 burnFee = MathHelper.max(ONE, quoteAmount / 1000);
quoteAmount = MathHelper.max(0, quoteAmount - burnFee);
```

`ONE = 10**18` represents $1 in X18 fixed-point. [1](#0-0) 

The `burnFee` floor is always at least `ONE` ($1). [2](#0-1) 

When `quoteAmount <= ONE` (i.e., the NLP being burned is worth ≤ $1), `burnFee = ONE >= quoteAmount`, so `quoteAmount = max(0, quoteAmount - burnFee) = 0`.

The function then conditionally skips the quote credit entirely:

```solidity
if (quoteAmount > 0) {
    spotEngine.updateBalance(QUOTE_PRODUCT_ID, txn.sender, quoteAmount);
    _applyNlpRebalance(spotEngine, nlpPools, nlpPoolRebalanceX18);
}
``` [3](#0-2) 

But the NLP debit is **unconditional** — it always executes regardless of whether `quoteAmount` is zero:

```solidity
spotEngine.updateBalance(NLP_PRODUCT_ID, txn.sender, -nlpAmount);
spotEngine.updateBalance(NLP_PRODUCT_ID, N_ACCOUNT, nlpAmount);
``` [4](#0-3) 

The user's NLP is fully burned; they receive nothing. There is no revert, no minimum output check, and no documentation warning about this behavior.

---

### Impact Explanation

A user burning NLP tokens whose total value is ≤ $1 loses their entire position with zero quote tokens returned. The state delta is:

- `spotEngine.balance[NLP_PRODUCT_ID][txn.sender]` decremented by `nlpAmount`
- `spotEngine.balance[NLP_PRODUCT_ID][N_ACCOUNT]` incremented by `nlpAmount`
- `spotEngine.balance[QUOTE_PRODUCT_ID][txn.sender]` unchanged (zero credit)

This is a direct, irreversible asset loss for the user. The corrupted invariant is: *a user who burns NLP must receive quote tokens proportional to the NLP value minus a reasonable fee*. When the fee floor exceeds the position value, this invariant is silently broken.

---

### Likelihood Explanation

This is reachable under realistic conditions:

1. A user accumulates a small residual NLP position (e.g., from partial burns, rounding, or price decline after a lock period).
2. The NLP oracle price (`oraclePriceX18`) is set such that the user's `nlpAmount * oraclePriceX18 <= 1e18` ($1).
3. The user signs and submits a `BurnNlp` transaction to exit their position.
4. The sequencer processes it via `EndpointTx.sol`, calling `clearinghouse.burnNlp()`. [5](#0-4) 

The 4-day NLP lock period (`NLP_LOCK_PERIOD`) means users may be unable to exit during adverse price movements, increasing the probability that a position's value falls below the $1 fee floor by the time it unlocks. [6](#0-5) 

---

### Recommendation

Add a minimum output guard before executing the NLP debit:

```solidity
require(quoteAmount > 0, "ERR_BURN_AMOUNT_TOO_SMALL");
```

This should be placed after line 504 and before line 511, so the transaction reverts rather than silently consuming the user's position. Alternatively, document explicitly that burns of NLP worth ≤ $1 result in total loss with zero quote returned.

---

### Proof of Concept

1. User holds 0.5 NLP tokens, oracle price = `1e18` ($1/NLP), so `quoteAmount = 0.5e18` ($0.50).
2. User signs `BurnNlp { nlpAmount: 0.5e18, ... }` and submits to the sequencer.
3. Sequencer calls `clearinghouse.burnNlp(txn, 1e18, ...)`.
4. `burnFee = max(ONE, 0.5e18 / 1000) = max(1e18, 5e14) = 1e18` ($1).
5. `quoteAmount = max(0, 0.5e18 - 1e18) = 0`.
6. `spotEngine.updateBalance(NLP_PRODUCT_ID, txn.sender, -0.5e18)` — NLP burned.
7. `if (quoteAmount > 0)` is false — no quote credited.
8. User loses $0.50 worth of NLP with zero return. No revert occurs. [7](#0-6)
