### Title
Missing destination-prefix guard in `processReceivedPacket` allows malicious counterparty to mint double-prefixed, unredeemable NFT vouchers — (`x/nft-transfer/keeper/packet.go`)

### Summary
`processReceivedPacket` unconditionally prepends the destination port/channel prefix to `data.ClassId` without first checking whether the classId already starts with that prefix. A malicious IBC counterparty can craft a packet whose `ClassId` already contains the destination prefix, causing the receiving chain to mint an NFT voucher under a double-prefixed class ID that has no corresponding escrowed NFT and can never be redeemed.

### Finding Description
In `processReceivedPacket` (`x/nft-transfer/keeper/packet.go` lines 154–157), when `isAwayFromOrigin` is true, the code computes:

```go
classPrefix := types.GetClassPrefix(packet.GetDestPort(), packet.GetDestChannel())
prefixedClassID := classPrefix + data.ClassId
```

There is no check that `data.ClassId` does not already start with `classPrefix`. `ValidateBasic()` (`x/nft-transfer/types/packet.go` lines 41–72) performs no such validation either. A malicious counterparty chain can commit a packet with `data.ClassId = "nft/channel-0/nativeClass"` (where `nft/channel-0` is the destination prefix), causing `prefixedClassID = "nft/channel-0/nft/channel-0/nativeClass"`. The resulting `voucherClassID = ibc/{sha256("nft/channel-0/nft/channel-0/nativeClass")}` is minted to the receiver with no backing escrow.

### Impact Explanation
The receiving chain mints NFT vouchers under a class ID that corresponds to no escrowed NFT on any chain. These vouchers:
1. Are permanently unredeemable — sending them back triggers a `TransferOwner` call against an empty escrow address, which fails.
2. Pollute the NFT module state with phantom class traces.
3. Can be minted to arbitrary receiver addresses, potentially deceiving users into believing they hold redeemable cross-chain NFTs.

### Likelihood Explanation
Any chain that has an open IBC channel with the victim chain can execute this attack by committing a crafted packet. No key compromise, governance action, or social engineering is required — only the ability to commit a packet on the counterparty chain, which is the normal capability of any counterparty chain operator.

### Recommendation
In `processReceivedPacket`, before prepending the destination prefix, check whether `data.ClassId` already starts with `classPrefix` and reject (or strip) accordingly. Alternatively, add a validation step in `ValidateBasic()` or `OnRecvPacket` that rejects any `ClassId` whose leading `port/channel/` segment matches the packet's destination port and channel.

### Proof of Concept
```go
// destPort="nft", destChannel="channel-0", sourcePort="nft", sourceChannel="channel-1"
// data.ClassId = "nft/channel-0/nativeClass"
//
// IsAwayFromOrigin("nft", "channel-1", "nft/channel-0/nativeClass")
//   => !HasPrefix("nft/channel-0/nativeClass", "nft/channel-1/") => true
//
// classPrefix = "nft/channel-0/"
// prefixedClassID = "nft/channel-0/nft/channel-0/nativeClass"  ← double-prefixed
//
// voucherClassID = ibc/{sha256("nft/channel-0/nft/channel-0/nativeClass")}
//   ≠ ibc/{sha256("nft/channel-0/nativeClass")}  ← what a legitimate escrow would use
//
// MintNFT(voucherClassID, tokenID, ...) succeeds → unbacked voucher exists on chain
// Redemption attempt: TransferOwner(voucherClassID, tokenID, escrowAddr, receiver) → fails (nothing escrowed)
```

func ParseClassTrace(rawClassID string) ClassTrace {
	classSplit := strings.Split(rawClassID, "/")

	if classSplit[0] == rawClassID {
		return ClassTrace{
			Path:        "",
			BaseClassId: rawClassID,
		}
	}

	return ClassTrace{
		Path:        strings.Join(classSplit[:len(classSplit)-1], "/"),
		BaseClassId: classSplit[len(classSplit)-1],
	}
}

func (nftpd NonFungibleTokenPacketData) ValidateBasic() error {
	if strings.TrimSpace(nftpd.ClassId) == "" {
		return newsdkerrors.Wrap(ErrInvalidClassID, "classId cannot be blank")
	}

	if len(nftpd.TokenIds) == 0 {
		return newsdkerrors.Wrap(ErrInvalidTokenID, "tokenId cannot be blank")
	}

	if len(nftpd.TokenIds) != len(nftpd.TokenUris) {
		return newsdkerrors.Wrap(ErrInvalidPacket, "tokenIds and tokenUris lengths do not match")
	}

	if strings.TrimSpace(nftpd.Sender) == "" {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "sender address cannot be blank")
	}

	// decode the sender address
	if _, err := sdk.AccAddressFromBech32(nftpd.Sender); err != nil {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "invalid sender address")
	}

	if strings.TrimSpace(nftpd.Receiver) == "" {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "receiver address cannot be blank")
	}

	// decode the receiver address
	if _, err := sdk.AccAddressFromBech32(nftpd.Receiver); err != nil {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "invalid receiver address")
	}
	return nil
}
