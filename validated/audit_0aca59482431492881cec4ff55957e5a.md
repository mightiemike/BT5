### Title
`ValidResourceBounds::L1Gas` / `AllResources` Canonicalization Collapse in Protobuf Deserialization Produces Wrong Transaction Hash - (File: crates/apollo_protobuf/src/converters/transaction.rs)

### Summary

The protobuf deserializer for `ResourceBounds` silently collapses `AllResources { l2_gas: 0, l1_data_gas: 0 }` into `ValidResourceBounds::L1Gas`. Because the two variants produce different hash preimages in `get_tip_resource_bounds_hash`, a transaction accepted by the gateway with the `AllResources` variant carries a hash that no peer can reproduce after P2P round-trip. The user's signature is over the gateway-computed hash; every peer rejects the deserialized form, and any block proposal containing the transaction is rejected by validators.

### Finding Description

**Serialization boundary — `TryFrom<protobuf::ResourceBounds> for ValidResourceBounds`** [1](#0-0) 

Line 427 silently defaults a missing `l1_data_gas` field to zero:

```rust
// TODO(Shahak): Assert data gas is not none once we remove support for 0.13.2.
let l1_data_gas = value.l1_data_gas.unwrap_or_default();
```

Lines 431–435 then apply a zero-test that is indistinguishable between `None` and `Some(0)`:

```rust
Ok(if l1_data_gas.is_zero() && l2_gas.is_zero() {
    ValidResourceBounds::L1Gas(l1_gas)
} else {
    ValidResourceBounds::AllResources(AllResourceBounds { l1_gas, l2_gas, l1_data_gas })
})
```

**Serialization path always emits `Some(0)` for zero fields** [2](#0-1) 

`From<ValidResourceBounds> for protobuf::ResourceBounds` always sets `l1_data_gas: Some(...)`. So a transaction originally stored as `AllResources { l1_gas: X, l2_gas: 0, l1_data_gas: 0 }` is serialized with `l1_data_gas: Some(0)`. On deserialization the zero-test fires and the variant becomes `L1Gas`.

**Hash preimage diverges between the two variants** [3](#0-2) 

`get_tip_resource_bounds_hash` includes `L1_DATA_GAS` in the Poseidon chain only for `AllResources`:

```
AllResources → Poseidon(tip, L1_GAS, L2_GAS, L1_DATA_GAS)   // H1
L1Gas        → Poseidon(tip, L1_GAS, L2_GAS)                 // H2 ≠ H1
```

**Gateway accepts with H1; peers reject with H2**

The gateway receives the transaction via JSON-RPC. The JSON deserializer (`TryFrom<DeprecatedResourceBoundsMapping>`) produces `AllResources` whenever `L1_DATA_GAS` is present in the map, even if zero: [4](#0-3) 

The gateway computes H1 and stores the transaction. The user signs H1.

When the transaction is propagated over P2P (mempool or consensus), it is deserialized via `TryFrom<protobuf::InvokeV3> for InvokeTransactionV3`, which calls the collapsing converter and produces `L1Gas`. The subsequent conversion to `RpcInvokeTransactionV3` then hard-fails: [5](#0-4) 

```rust
resource_bounds: match value.resource_bounds {
    ValidResourceBounds::AllResources(bounds) => bounds,
    _ => {
        return Err(StarknetApiError::OutOfRange { string: "resource_bounds".to_string() });
    }
},
```

The error is surfaced as `DEPRECATED_RESOURCE_BOUNDS_ERROR`: [6](#0-5) 

**Consensus path**

The same `TryFrom<protobuf::InvokeV3> for InvokeTransactionV3` is used when validators deserialize a `ConsensusTransaction` sent by the proposer: [7](#0-6) 

Validators cannot reconstruct the transaction and reject the block proposal.

### Impact Explanation

A user who submits an `InvokeV3` transaction with `AllResourceBounds { l1_gas: X, l2_gas: 0, l1_data_gas: 0 }` (a valid combination accepted by the stateless validator) receives a gateway-assigned hash H1. The signature covers H1. After any P2P hop — mempool propagation or consensus proposal — the transaction fails deserialization on every peer. If the proposer includes the transaction in a block, all validators reject the proposal. The transaction is permanently unsequenceable despite being admitted by the gateway.

impl TryFrom<protobuf::ResourceBounds> for ValidResourceBounds {
    type Error = ProtobufConversionError;
    fn try_from(value: protobuf::ResourceBounds) -> Result<Self, Self::Error> {
        let Some(l1_gas) = value.l1_gas else {
            return Err(missing("ResourceBounds::l1_gas"));

        let Some(l2_gas) = value.l2_gas else {
            return Err(missing("ResourceBounds::l2_gas"));

            ValidResourceBounds::L1Gas(l1_gas)

            ValidResourceBounds::AllResources(AllResourceBounds { l1_gas, l2_gas, l1_data_gas })
        })
    }

pub fn get_tip_resource_bounds_hash(
    resource_bounds: &ValidResourceBounds,
    tip: &Tip,
) -> Result<Felt, StarknetApiError> {
    let l1_resource_bounds = resource_bounds.get_l1_bounds();
    let l2_resource_bounds = resource_bounds.get_l2_bounds();

    // L1 and L2 gas bounds always exist.
    // Old V3 txs always have L2 gas bounds of zero, but they exist.
    let mut resource_felts = vec![
        get_concat_resource(&l1_resource_bounds, L1_GAS)?,
        get_concat_resource(&l2_resource_bounds, L2_GAS)?,
    ];

    // For new V3 txs, need to also hash the data gas bounds.
    resource_felts.extend(match resource_bounds {
        ValidResourceBounds::L1Gas(_) => vec![],
        ValidResourceBounds::AllResources(all_resources) => {
            vec![get_concat_resource(&all_resources.l1_data_gas, L1_DATA_GAS)?]
        }
    });

    Ok(HashChain::new().chain(&tip.0.into()).chain_iter(resource_felts.iter()).get_poseidon_hash())
}

impl TryFrom<DeprecatedResourceBoundsMapping> for ValidResourceBounds {
    type Error = StarknetApiError;
    fn try_from(
        resource_bounds_mapping: DeprecatedResourceBoundsMapping,
    ) -> Result<Self, Self::Error> {
        if let (Some(l1_bounds), Some(l2_bounds)) = (
            resource_bounds_mapping.0.get(&Resource::L1Gas),
            resource_bounds_mapping.0.get(&Resource::L2Gas),
        ) {
            match resource_bounds_mapping.0.get(&Resource::L1DataGas) {
                Some(data_bounds) => Ok(Self::AllResources(AllResourceBounds {
                    l1_gas: *l1_bounds,
                    l1_data_gas: *data_bounds,
                    l2_gas: *l2_bounds,
                })),
                None => {
                    if l2_bounds.is_zero() {
                        Ok(Self::L1Gas(*l1_bounds))
                    } else {
                        Err(StarknetApiError::InvalidResourceMappingInitializer(format!(
                            "Missing data gas bounds but L2 gas bound is not zero: \
                             {resource_bounds_mapping:?}",
                        )))
                    }
                }
            }
        } else {
            Err(StarknetApiError::InvalidResourceMappingInitializer(format!(
                "{resource_bounds_mapping:?}",
            )))
        }
    }
