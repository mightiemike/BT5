### Title
Attacker-Controlled `data.ClassId` Flips `IsAwayFromOrigin` Direction Check, Enabling Theft of Escrowed NFTs — (`x/nft-transfer/keeper/packet.go`)

---

### Summary

`processReceivedPacket` determines the transfer direction (mint vs. unescrow) by calling `IsAwayFromOrigin` with the attacker-controlled `data.ClassId`. Because `ValidateBasic` does not verify that `ClassId` is consistent with the packet's source port/channel, a malicious counterparty chain can craft a `ClassId` that starts with `sourcePort/sourceChannel/`, flipping the direction check and forcing the unescrow branch. If the receiving chain holds legitimately escrowed NFTs on that channel, the attacker can drain them.

---

### Finding Description

**`IsAwayFromOrigin` logic** — `x/nft-transfer/types/trace.go` lines 49–52:

```go
func IsAwayFromOrigin(sourcePort, sourceChannel, fullClassPath string) bool {
    prefixClassID := GetClassPrefix(sourcePort, sourceChannel)
    return !strings.HasPrefix(fullClassPath, prefixClassID)
}
```

It returns `false` (i.e., "back to origin → unescrow") whenever `fullClassPath` starts with `sourcePort/sourceChannel/`. [1](#0-0) 

**Call site in `processReceivedPacket`** — `x/nft-transfer/keeper/packet.go` line 148:

```go
isAwayFromOrigin := types.IsAwayFromOrigin(packet.GetSourcePort(), packet.GetSourceChannel(), data.ClassId)
```

`packet.GetSourcePort()` and `packet.GetSourceChannel()` are IBC-core-set and cannot be forged. `data.ClassId` is **packet application data** — fully controlled by the sending chain (the attacker). [2](#0-1) 

**`ValidateBasic` does not guard this** — `x/nft-transfer/types/packet.go` lines 41–72:

`ValidateBasic` only checks that `ClassId` is non-blank, `TokenIds` is non-empty, lengths match, and addresses are valid bech32. It performs **no check** that `ClassId` is consistent with the packet's source port/channel. [3](#0-2) 

**Unescrow branch** — `x/nft-transfer/keeper/packet.go` lines 186–201:

```go
} else {
    unprefixedClassID := types.RemoveClassPrefix(packet.GetSourcePort(),
        packet.GetSourceChannel(), data.ClassId)
    voucherClassID := types.ParseClassTrace(unprefixedClassID).IBCClassID()
    for _, tokenID := range data.TokenIds {
```

`escrowAddress` is `GetEscrowAddress(packet.GetDestPort(), packet.GetDestChannel())` — the escrow address on the **receiving** chain for that channel. [4](#0-3) 

---

### Impact Explanation

**Concrete theft scenario:**

1. Chain B (victim) holds native NFT class `"nativeNFT"`, tokenID `"token1"`.
2. Chain B legitimately sends it to chain A via port `"nft"`, channel `"channel-0"` (chain B side). The NFT is escrowed at `GetEscrowAddress("nft", "channel-0")` on chain B.
3. Attacker (controlling chain A, port `"nft"`, channel `"channel-1"`) crafts a packet to chain B:
   - `data.ClassId = "nft/channel-1/nativeNFT"` — starts with `"nft/channel-1/"` (chain A's source prefix)
   - `data.TokenIds = ["token1"]`
   - `data.Receiver = attacker_address_on_B`
4. On chain B, `IsAwayFromOrigin("nft", "channel-1", "nft/channel-1/nativeNFT")` → `false` (prefix match).
5. Unescrow branch executes:
   - `unprefixedClassID = "nativeNFT"`
   - `voucherClassID = "nativeNFT"`
   - `escrowAddress = GetEscrowAddress("nft", "channel-0")` ← this address **does** own `"nativeNFT"/"token1"`
   - `TransferOwner(ctx, "nativeNFT", "token1", escrowAddress, attacker)` **succeeds**

The attacker receives an NFT they never legitimately redeemed. The escrow invariant is broken: an NFT is released without a corresponding burn on the counterparty. [5](#0-4) 

---

### Likelihood Explanation

The attacker must control a chain that has an established IBC channel with the victim chain. This is a realistic threat model in a multi-chain ecosystem — any chain operator with a live IBC channel can exploit this. No governance, key compromise, or social engineering is required. The only precondition is that the victim chain has previously escrowed NFTs on the targeted channel (a normal operational state).

---

### Recommendation

The direction check must be anchored to the **canonical class path** stored on-chain, not to attacker-supplied `data.ClassId`. Specifically:

1. In `processReceivedPacket`, verify that if `data.ClassId` starts with `sourcePort/sourceChannel/`, the receiving chain actually has a class trace record for the stripped path — or reject the packet if the prefix is inconsistent with the expected direction.
2. Alternatively, enforce in `ValidateBasic` (or in `OnRecvPacket` before calling `processReceivedPacket`) that `data.ClassId` does **not** start with the packet's own `sourcePort/sourceChannel/` prefix unless the receiving chain can confirm it previously issued that voucher.
3. The safest fix mirrors ICS-20's approach: on receive, always treat the packet as "away from origin" (mint) unless the `ClassId` carries a prefix that was added by **this** chain (i.e., `destPort/destChannel/`), not the source chain's prefix.

```go
func (k Keeper) processReceivedPacket(ctx sdk.Context, packet channeltypes.Packet,
	data types.NonFungibleTokenPacketData,
) error {
	receiver, err := sdk.AccAddressFromBech32(data.Receiver)
	if err != nil {
		return err
	}

	isAwayFromOrigin := types.IsAwayFromOrigin(packet.GetSourcePort(), packet.GetSourceChannel(), data.ClassId)

	// create the escrow address for creating denom and minting nft
	escrowAddress := types.GetEscrowAddress(packet.GetDestPort(), packet.GetDestChannel())

	if isAwayFromOrigin {
		// since SendPacket did not prefix the classID, we must prefix classID here
		classPrefix := types.GetClassPrefix(packet.GetDestPort(), packet.GetDestChannel())
		// NOTE: sourcePrefix contains the trailing "/"
		prefixedClassID := classPrefix + data.ClassId

		// construct the class trace from the full raw classID
		classTrace := types.ParseClassTrace(prefixedClassID)
		if !k.HasClassTrace(ctx, classTrace.Hash()) {
			k.SetClassTrace(ctx, classTrace)
		}

		voucherClassID := classTrace.IBCClassID()

		if !k.nftKeeper.HasDenomID(ctx, voucherClassID) {
			if err := k.nftKeeper.IssueDenom(ctx, voucherClassID, voucherClassID, "", data.ClassUri, escrowAddress); err != nil {
				return err
			}
		}
		sdkCtx := sdk.UnwrapSDKContext(ctx)
		sdkCtx.EventManager().EmitEvent(
			sdk.NewEvent(
				types.EventTypeClassTrace,
				sdk.NewAttribute(types.AttributeKeyTraceHash, classTrace.Hash().String()),
				sdk.NewAttribute(types.AttributeKeyClassID, voucherClassID),
			),
		)

		for i, tokenID := range data.TokenIds {
			if err := k.nftKeeper.MintNFT(ctx, voucherClassID, tokenID, "", data.TokenUris[i], "", escrowAddress, receiver); err != nil {
				return err
			}
		}
	} else {
		// If the token moves in the direction of back to origin,
		// we need to unescrow the token and transfer it to the receiver

		// we should remove the prefix. For example:
		// p6/c6/p4/c4/p2/c2/nftClass -> p4/c4/p2/c2/nftClass
		unprefixedClassID := types.RemoveClassPrefix(packet.GetSourcePort(),
			packet.GetSourceChannel(), data.ClassId)

		voucherClassID := types.ParseClassTrace(unprefixedClassID).IBCClassID()
		for _, tokenID := range data.TokenIds {
			if err := k.nftKeeper.TransferOwner(ctx,
				voucherClassID, tokenID, escrowAddress, receiver); err != nil {
				return err
			}
		}
	}

	return nil
}
```

```go
func (nftpd NonFungibleTokenPacketData) ValidateBasic() error {
	if strings.TrimSpace(nftpd.ClassId) == "" {
		return newsdkerrors.Wrap(ErrInvalidClassID, "classId cannot be blank")
	}

	if len(nftpd.TokenIds) == 0 {
		return newsdkerrors.Wrap(ErrInvalidTokenID, "tokenId cannot be blank")
	}

	if len(nftpd.TokenIds) != len(nftpd.TokenUris) {
		return newsdkerrors.Wrap(ErrInvalidPacket, "tokenIds and tokenUris lengths do not match")
	}

	if strings.TrimSpace(nftpd.Sender) == "" {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "sender address cannot be blank")
	}

	// decode the sender address
	if _, err := sdk.AccAddressFromBech32(nftpd.Sender); err != nil {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "invalid sender address")
	}

	if strings.TrimSpace(nftpd.Receiver) == "" {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "receiver address cannot be blank")
	}

	// decode the receiver address
	if _, err := sdk.AccAddressFromBech32(nftpd.Receiver); err != nil {
		return newsdkerrors.Wrap(sdkerrors.ErrInvalidAddress, "invalid receiver address")
	}
	return nil
}
```
